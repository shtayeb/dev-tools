Get started

    Endpoint

    https://api.shahryartayeb.com/generate-image

The application is an image generation api that enables users to create custom images. The app provides a /generate-image route, When a user makes a GET request to this route with the desired width and height, the app generates a banner image with the specified dimensions and adds a text. The generated banner image is returned to the user in a JPEG format.

To use this API, you do not need an API key. Please contact us at shtb@shahryartayeb.com for more information

get characters


# Here is a curl example
curl \
-X POST https://api.shahryartayeb.com/generate-image\
-F 'text=Hello World' \

To generate your image, you need to make a GET call to the following url :
https://api.shahryartayeb.com/generate-image

QUERY PARAMETERS

Field Type Description
text String Text that you want to add to the image.
width Number (optional) Width of the image. default is 1920
height Number (optional) Height of the image. default is 1080
border_color String (optional) the top and bottom border color. default is 'purple'
border_width Number (optional) width of the top and bottom border. default is 40px

Errors

The Westeros API uses the following error codes:

Error Code Meaning
400 400 Bad Request: If the width or height parameter is missing or invalid. Some parameters are missing. This error appears when you don't pass every mandatory parameters.
500 500 Internal Server Error: If there is an error generating the image.
